# team-project


## Instalacja backendu na WSL ubuntu:
```
curl --proto '=https' --tlsv1.3 [https://sh.rustup.rs](https://sh.rustup.rs/) -sSf | sh
sudo apt update
sudo apt upgrade
sudo apt install cargo rustup pkg-config libssl-dev
rustup update
cd /mnt/c/Users/.../.../team-project/backend
cargo build
cargo run
```
